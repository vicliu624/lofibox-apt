Documentation

LoFiBox Zero 文档

这份官网文档是给普通用户、发行测试者和自动化使用者读的入口。这份文档会把产品边界、安装路径和主要命令讲清楚。

Get Started

安装预览版、第一次运行、扫描媒体库、播放音乐、诊断环境。

GUI

小屏图形界面的菜单、媒体库、搜索、Now Playing、歌词和 EQ。

TUI

终端 Dashboard、Lyrics、Spectrum、Queue、EQ，以及字符集和主题。

CLI

人类命令、机器输出、runtime 命令、source/remote、metadata/lyrics、稳定退出码。

远程媒体库

支持哪些远程源，如何配置 profile、探测能力、浏览目录、解析流。

元数据与歌词

ID3/metadata、lyrics、artwork、fingerprint 的补齐、缓存、写回边界。

产品边界

LoFiBox Zero 的核心原则是：事实层、runtime 投影层、GUI/TUI/CLI 展示层必须分开。 播放状态、队列、EQ、远程连接、歌词和媒体库事实属于 runtime 与 core； GUI/TUI/CLI 只读取投影并提交受控命令。

入口 定位 适合场景
GUI 低分辨率图形产品界面 Cardputer Zero、Raspberry Pi 小屏、Linux 桌面窗口
TUI 终端原生产品界面 SSH、服务器、无桌面环境、开发者和创作者终端
CLI 一次性命令与机器接口 脚本、systemd、测试、MCP/Agent、故障诊断
