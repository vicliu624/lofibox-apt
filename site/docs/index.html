Product Boundaries

LoFiBox Zero follows one core rule: the fact layer, runtime projection layer, and GUI/TUI/CLI presentation layers stay separate. Playback state, queue, EQ, remote connections, lyrics, and library facts belong to the runtime and core domains; GUI/TUI/CLI read projections and submit controlled commands.

TUI Native terminal product interface SSH, servers, headless systems, developer and creator terminals
